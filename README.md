# HR Attrition Analysis Project

## 📊 Overview
This project analyzes employee attrition using HR data to identify key factors influencing employee turnover.

## 🛠 Tools Used
- Excel (Data Analysis)
- Tableau (Dashboard)
- Python (Pandas, Scikit-learn)

## 🔍 Key Insights
- Employees working overtime have 3x higher attrition (30% vs 10%)
- Lower salary groups show higher attrition
- Low job satisfaction leads to higher turnover

## 🤖 Machine Learning
- Built Logistic Regression model
- Identified Overtime as strongest predictor
